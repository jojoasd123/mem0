# Vault: 记忆同步协议

**类型**: 永久保留 (No Decay)
**重要性**: Critical

## 云端 ↔️ 本地 同步机制

### 架构
- **云端芋圆**: 腾讯云服务器运行
- **本地芋圆**: 主人笔记本运行
- **同步方式**: Git 仓库

### 分支策略
- `main` - 云端芋圆的主分支
- `local` - 本地芋圆的工作分支

### 同步流程
1. 本地芋圆提交更改到 `local` 分支
2. 云端芋圆每天凌晨 3:00 执行合并任务
3. 合并 `local` → `main`
4. 云端芋圆拉取最新 `main`

### 冲突处理
- 以时间戳为准，较新的更改优先
- 重要记忆手动合并
